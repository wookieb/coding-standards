# Coding standard

This is a set of coding standard useful to make quick and consistent decision within a team.

The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT",
"SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this document are to be interpreted as described
in [RFC 2119](https://datatracker.ietf.org/doc/html/rfc2119)
